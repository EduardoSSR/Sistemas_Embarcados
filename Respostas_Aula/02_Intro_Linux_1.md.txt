1. Por que o Linux recebeu esse nome?

[Linux foi o nome dado ao núcleo de sistema operacional criado por Linus Torvalds. ]
2. O que são daemons?
[Daemons são é basicamente um programa executado como um processo em plano de fundo, o qual não está sobre controle direto de um usuário interativo.]

3. O que é o shell?


[A palavra SHELL , em computação, é considerado a camada externa entre o usuário e o kernel (núcleo) de um sistema operacional. Mais usualmente usados para
denominar os programa com os quais os usuários de sistemas baseados em Unix interagem com o kernel]
4. Por que é importante evitar executar o terminal como super-usuário?

[Porque o usuário passa a ter permissões de administrador do sistema operacional e pode deletar ou desconfigurar algo importante]
5. Qual botão do teclado completa o que o usuário escreve no terminal, de acordo com o contexto?


[ Tecla Tab.]
6. Quais botões do teclado apresentam instruções escritas anteriormente?
[A tecla de seta para cima]
